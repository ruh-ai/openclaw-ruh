# Source Library

This folder contains the raw source material for the project knowledge base.

## Structure
- `Platform Core`: PRD, system design, and pack specification.
- `Reference Packs`: reusable pack definitions for target domains.
- `Tenant Overlays`: overlay kits and machine-readable overlay manifests.
- `Pilot Delivery`: pilot runbook and rollout material.
- `Go To Market`: commercial and proposal documents.

## Usage
- Treat the files here as source of truth.
- Use the generated notes under `Knowledge Base/Documents/` when you want source-backed summaries and cross-links.
- Use `Knowledge Base/Project Knowledge/` when you want the synthesized working understanding.
